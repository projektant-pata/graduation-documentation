Pole

  • lineární struktura
  • prvky jsou pevně za sebou
  • pevný počet prvků, při změně nutnost vytvoření nového pole Arrays.copyOf

Složitosti

  • Přístup k prvku:
    • vždycky O(1)
  • Vkládání a mazání prvku:
    • vždycky O(n)

Využití

  • pokud máme konstantní velikost prvků a minimum úprav, využije se rychlé čtení

Seznamy (List)

  • lineární struktura

ArrayList

  • dynamické pole

Složitosti

  • Přístup k prvku:
    • vždycky O(1)
  • Vkládání a mazání prvku:

Složitosti

  • Přístup k prvku:

  • Vkládání a mazání prvku:
    • vždycky O(1)

Využití

  • pokud víme, že se počet prvků bude tak často měnit
  • např. implementace fronta, zásobník

Množiny (Set)

  • nemůžou být duplicity

HashSet

  • založeno na hashovací tabulce
  • neuchovává pořadí

Složitosti

  • Přístup k prvku:
    • vždycky O(1)
  • Vkládání a mazání prvku:
    • vždycky O(1)

Využití

  • pro rychlé hledání a úpravy, když není potřeba pořadí a je třeba zabránit duplicitám

Složitosti

  • Přístup k prvku:
    • vždycky O(log n)
  • Vkládání a mazání prvku:

Využití

Složitosti

  • Přístup k prvku:

  • Vkládání a mazání prvku:
    • vždycky O(log n)

Comparable vs. Comparator

VlastnostComparatorComparable
Kde je:Mimo třídu (samostatný objekt)Uvnitř třídy objektu (implements)
Metoda:compare(obj1, obj2)compareTo(this, other)
Počet:Můžeš jich mít nekonečně mnohoJen jedno výchozí řazení
Volání:Collections.sort(list, comparator)Collections.sort(list)

Ukázka Comparator

class VekComparator implements Comparator<Student> {
    @Override
    public int compare(Student s1, Student s2) {
        return s1.vek.compareTo(s2.vek);
    }
}

// Použití pro seřazení např. ArrayList
Collections.sort(studenti, new JmenoComparator());

//Použití pro porovnání 2 objektů
Student petr = new Student("Petr", 22);
Student jana = new Student("Jana", 22);

// Vytvoříš instanci "rozhodčího" (komparátoru)
JmenoComparator podleJmena = new JmenoComparator();

// "Rozhodčí" porovná dva objekty
int vysledek = podleJmena.compare(petr, jana); //vysledek == 0

Ukázka Comparable

class Student implements Comparable<Student> {
    String jmeno;
    int vek;

    public Student(String jmeno, int vek) {
        this.jmeno = jmeno;
        this.vek = vek;
    }

    @Override
    public int compareTo(Student jinyStudent) {
        // Vrací: záporné (menší), 0 (stejné), kladné (větší)
        return this.vek - jinyStudent.vek; 
    }
}

// Použití pro seřazení např. ArrayList
Collections.sort(studenti);

Student petr = new Student("Petr", 25);
Student jana = new Student("Jana", 22);

int vysledek = petr.compareTo(jana); //vysledek > 0
