2. Řídící struktury, datové typy

Řídící struktury

  • určuje, jakým způsobem se bude vykonávat posloupnost příkazů

1. Posloupnost

  • nejjednodušší typ
  • příkazy se provádí jeden za druhým (ve stejném pořadí, jak jsou napsány)
  • spojují se přes {} (v Javě, Python je třeba nemá)
void main(){
	int i = 0; //prvni, definuje i
	i++; //druhy, prida i + 1
	//
}

2. Větvení

  • = if, else if, else
  • program vybírá mezi cestami podle podmínek (vybere, pokud true)
if(i == 0){
	//
}else if(i == 1){
	//
}else {
	//
}

3. Switch

  • cool else if
  • pokud se case se rovná s podmínkou, vykoná se case
  • pokud se aktivuje case, pokračuje neustále (tomuhle předejdeme s break)
  • default se provede vždy (neplatí, pokud je switch ukončen break)
switch(i){
	case 0:
		//
		break;
	case 1:
		//
		break;
	default:
		//
}

4. Smyčky (cykly)

  • opakují určitý blok kódu (dokud je podmínka)

For

  • pevný počet opakování 
for(int i = 0; i < 10; i++) {
  // opakovaný kód
}

While

  • opakuje se, dokud je podmínka pravdivá
while(podminka){
	//
}

Do-while

  • while, ale aspoň jednou se provede
  • jediný loop s ;
do{
	//
}while(podminka);

Řídící příkazy

Break

  • ukončí nejbližší cyklus nebo switch, ve kterém je vnořený

Continue

  • přeskočí aktuální iteraci cyklu (potom normálně pokračuje)

Return

  • ukončí metodu a popřípadně vrátí návratovou hodnotu

int i = 0;
while(true){
	if(i == 4) // pokud bude i 4, tak se přičte +2
		i+=2;
		continue;
	else if (i == 7) //pokud bude i 7, tak se ukončí cyklus
		break;
	i++;
}
