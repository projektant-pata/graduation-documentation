Regulární výrazy

= regex

  • vyhledání textu dle vzoru
  • 2 způsoby využití v Javě:
    • Metody třídy String
    • Pattern a Matcher

Metody třídy String

  • používá se pro jednorázové použití

matches() - vrací boolean, pokud se text shoduje s regexem

replaceAll() - nahradí všechny části řetězce shodné s regexem

Pattern a matcher

  • používá se při několikanásobném užití jednoho regexu

Pattern

  • vytvoří se pattern pro regulární výraz

compile(String regex) - vytvoří Pattern (statická metoda)

matcher(String txt) - vytvoří Matcher

Matcher

  • porovnává text s Pattern a hledá shody

find(): boolean - vrací true, pokud je shoda v textu

matches(): boolean - vrací true, pokud je shodný celý text

Ukázka

StringBuilder

  • StringBuffer má stejnou syntax
StringBuilder sb = new StringBuilder("je"); //"je"
 
sb.append(" hořký!"); //"je hořký!"

sb.insert(0, "Život "); //"Život je hořký!"

sb.replace(9, 15, "boj"); //"Život je boj"

sb.delete(0, 9); // "boj"

sb.reverse(); // "job"

sb.setLength(0); // ""

Hledání pomocí: metody třídy String x Pattern a Matcher

// Parametrz
String text = "Život je hořký. Bohudík!";
String slovo = "hořký";

int pozice = text.indexOf(slovo);

if (pozice != -1) {
    IO.println("Pozice: " + pozice); // 9
    IO.println("Konec: " + (pozice + slovo.length())); // 14
}

Pattern pattern = Pattern.compile(slovo );
Matcher matcher = pattern.matcher(text);

if (matcher.find()) {
    IO.println("Pozice: " + matcher.start()); // 9
    IO.println("Konec: " + matcher.end()); // 14
}

Cvičení na regex

Základní
IČO

ZADÁNÍ

Udělej regex na IČO

  • TEMPLATE
    import java.util.regex.Pattern;
import java.util.regex.Matcher;

public class EmailValidator {
    private static final String REGEX = "";
    private static final Pattern PATTERN =
            Pattern.compile(REGEX, Pattern.CASE_INSENSITIVE);

    public static boolean isValidExample(String example) {
        if (example == null) {
            return false;
        }
        Matcher matcher = PATTERN.matcher(example);
        return matcher.matches();
    }

    public static void main(String[] args) {
        String[] examples = {
                "69855706",// valid
                "22360808",// valid
                "12345678",// valid
                "jednadvatrictyripetsestsemdosm", // invalid
                "012345678", // invalid
                "67 67 67 67" // invalid
        };

        for (String example : examples) {
            System.out.println(example + ": " + isValidExample(example)); // pro mrdky co nemaji javu 25 je sout :(
        }
    }
}
