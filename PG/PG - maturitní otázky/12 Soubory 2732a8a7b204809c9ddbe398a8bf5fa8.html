12. Soubory

Typy souborů

  • při práci je důležitý znát typ souboru
  • každý typ má jiný styl ukládání a čtení dat
  • po práci se souborem uzavřít stream (uvolnění prostředků)
  • výjimky IOException, FileNotFoundException

File

  • používa se k:
    • zkontrolování existence souboru
    • získání metadat (velikost, typ..)
    • přejmenování souboru
    • vytvoření souboru
    • smazání souboru

Textové soubory

  • data jsou uložena jako text ⇒ čitelné pro člověka
  • jsou kódována (nejčastější je UTF-8)
  • každý znak je sekvence bajtů
  • formáty .csv, .txt, .json

BufferedReader

  • čte z textových souborů
  • “lepší” FileReader, protože využívá cache

BufferedWriter

  • zapisuje do textových souborů
  • “lepší” FileWriter, protože využívá cache

Binární (bytové) soubory

  • data jsou uložena v nečíslované formě (série bajtů) ⇒ nejsou čitelné pro člověka
  • formáty .jpg, .mp3, .pdf

BufferedInputStream

  • čte z binárních souborů
  • “lepší” FileInputStream, protože využívá cache

BufferedOutputStream

  • zapisuje do binárních souborů
  • “lepší” FileOutputStreamr, protože využívá cache

Práce s prostředky

  • = try-with-resources (try s ())
  • platí jen pro objekty, co implementují Closable nebo AutoClosable
  • všechno v () se automaticky uzavře po skončení try
//automaticky se zavola close() na resource po dokonceni
try(Resource resource = new ResourceType()){
	//práce s prostředkem
} catch (Exception e){
	//zpracovani
}
