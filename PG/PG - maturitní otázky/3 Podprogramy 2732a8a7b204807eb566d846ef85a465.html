3. Podprogramy

  • předdefinovaná pojmenovaná část kódů, která vykonává určitou činnost
  • může být znovupoužitelný
  • patří sem:
    • Metoda
    • Funkce

Metoda x Funkce

VlastnostFunkceMetoda
VazbaNezávisláVáže se na objekt nebo třídu
VoláníPřímo Přes třídu či instanci
Přístup k datůmS předanými datyS předanými daty + atributy třídy nebo objektu
Příklad jazykůC, PythonJava, C#, Python

Deklarace podprogramu

Modifikátor dostupnosti

  • pouze u metod (nedostupná funkce je stejně dobrá jako žádná)
  • několik typů
ModifikátorPřístupnostKdo má přístup

protectedChráněnyTřídy ve stejném package (složce) + potomci kdekoliv
privateSoukromýPouze uvnitř třídy, která ho obsahuje
FriendlyTřídy ve stejném package (složce)

Hlavička

Název

  • jak se jmenuje podprogram
  • výstižně pojmenovávat metody!!

Vstupní parametry

  • zadá se typ a název parametru (tím se uvnitř podprogramu vytvoří proměnná)

Návratový typ

  • co podprogram vrací
  • může vracet
    • primitivní datové typy
    • referenční
    • nic (void)

Přetěžování podprogramů

  • 2 stejně nazvané programy s jinou signaturou

Signatura

  • “unikátnost metody”
  • aby mohla být vytvořena, musí mít min. 1 unikátní vlastnost
    • název
    • vstupní parametry
      • počet
      • typ
      • pořadí

Rekurzivní podprogramy

  • podprogram volá sám sebe před tím, než bylo dokončeno předchozí volání
  • např. při procházení stromu (větví) nebo u fibonacciho posloupnosti
  • musí nějak skončit (má 2 typy větví - jedna končí, druhá posílá dál)

Volání podprogramu

  1. zavolá se podprogram
  1. uloží se do Stack (část paměti, kam se ukládají rámce podprogramů)
  1. po ukončení se odstraní ze Stack

Ukončení podprogramu

  • return (vrací hodnotu, u void bez parametru = čistě ukončí metodu)
  • throw new (vyhodí výjimku)
  • doběhnutí na konec

Ukázka

Příklad funkce v jazyce C

#include <stdio.h>

// Funkce: samostatná, mimo třídu
int secti(int a, int b) {
    return a + b;
}

int main() {
    int vysledek = secti(5, 3);  // volání funkce
    printf("Výsledek: %d\n", vysledek);
    return 0;
}

Příklad metody v Javě

public class Kalkulacka {

    // Metoda: je uvnitř třídy, vázaná na třídu
    public static int secti(int a, int b) {
        return a + b;
    }

    // Hlavní metoda – vstupní bod programu
    public static void main(String[] args) {
        int vysledek = secti(5, 3);  // volání metody
        System.out.println("Výsledek: " + vysledek);
    }
}

Frajer

Čím se dá nahradit rekurzní metoda?

Cyklem - řeší podobný věci

rekurze přidává čitelnost, ale horší na paměť (všechny volané programy jsou ukládané ve stacku)

Jaký je rozdíl mezi funkcí a procedurou?
  • procedura nemá výstup (něco jako void v Javě)
  • výskyt např. v SQL
Co to jsou instanční, statické metody a inicializační, statické bloky? (v kontextu OOP)

Metody

  • instanční - je pod objektem
  • statická - je pod třídou

Bloky

  • inicializační - spustí se při vytváření instance
  • statický - spustí se jednou při načtení třídy
