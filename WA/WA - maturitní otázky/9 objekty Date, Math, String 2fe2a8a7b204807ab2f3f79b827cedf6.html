9. objekty Date, Math, String

Date

  • objekt Date se uchovává jako počet milisekund od 1. ledna 1970 UTC

Způsoby vytvoření

const ted = new Date(); // Aktuální čas
const datum = new Date(2024, 2, 15, 14, 30, 45, 500); // 15. leden 2024 (měsíce jsou 0-11!)
const zMilisekund = new Date(1700000000000); // Z "goofy" milisekund
const zeStringu = new Date("2024-01-15"); // Z ISO stringu

Metody

  • převážně gettery a settery
  • !! nepoužívat getYear, setYear, ale getFullYear, setFullYear

Gettery

  • getFullYear() – rok (např. 2024)
  • getMonth() – měsíc (0–11) !! 0 = Leden
  • getDate() – den v měsíci (1–31)
  • getDay() – den v týdnu (0–6) !! 0 = Neděle
  • getHours() – hodiny (0–23)
  • getMinutes() – minuty (0–59)
  • getSeconds() – sekundy (0–59)
  • getMilliseconds() – milisekundy (0–999)
  • getTime() – počet milisekund od 1. 1. 1970 (timestamp)

Settery

  • setFullYear(rok, [měsíc], [den])
  • setMonth(měsíc, [den])
  • setDate(den)
  • setHours(hodiny, [min], [sec], [ms])
  • setMinutes(min, [sec], [ms])
  • setSeconds(sec, [ms])
  • setMilliseconds(ms)
  • setTime(timestamp) – nastaví datum natvrdo podle počtu ms od 1970

Math

  • statický objekt (nelze deklarovat)
  • obsahuje matematické operace a hodnoty

Nejužitečnější atributy

.PI - číslo π (3.14…)

.SQRT2 - odmocnina ze 2

.LN10 - logaritmus 10

Nejužitečnější metody

.round(int) - zaokrouhlí

.cell(int) - zaokrouhlí nahoru

.floor(int) - zaokrouhlí dolů

.sqrt(x) - odmocnina

String

  • immutable (neměnný)

    ⇒ vše vrací nový String (změna v proměnné, metody..)

Užitečné atributy

.length - délka

Užitečné metody

.charAt(int) - char na indexu

.concat(string, text2) - spojí 2 stringy a dá mezi ně spojku

.slice(base, offset) - vrací string od indexu po index

.toLowerCase() + .toUppercase - vrací string s malými/velkými znaky

.trim() - smaže bílé znaky na začátku a na konci stringu

.localeCompare(string2, locales) - vrací 1 (true), -1 (false), 0 (==) + volitelná možnost nastavení locales (pro české znaky “cs”)

Bajer Hund

Co se stane, když dám třeba den 32 do date?

Date je chytrý a převede si to na další měsíc

// 32. leden 2024 -> Autokorekce na 1. únor 2024
const opraveno = new Date(2024, 0, 32); 
console.log(opraveno.toLocaleDateString()); // "1. 2. 2024"
Jak zjistím přestupný rok?

Metoda 1: matematicky

  • rok % 4 === 0 v podmínce

Metoda 2: využití auto korekce v Date

  • pokud bude stále únor, je přestupný, pokud ne, není
function jePrestupny(rok) {
    // 29. den v měsíci 1 (únor)
    return new Date(rok, 1, 29).getMonth() === 1;
}
