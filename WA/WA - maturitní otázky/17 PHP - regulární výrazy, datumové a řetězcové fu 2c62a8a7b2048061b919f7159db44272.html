Metody

preg_match(string $patt, string $subj, array &$matches, int $flags, int offset)

  • vrátí 1 při shodě, 0 při neshodě a false při chybě
  • parametry
    • pattern (regex)
      • povinný
    • subject (text)
      • povinný
    • matches
      • uloží shody (napřed [0] - všechen text, pak shody jednotlivých podmínek (závorek)
      • matches[0] = všechen text, co měl shodu
      • matches[1] = text odpovídající první závorce (subpatternu)
      • matches[2] = text odpovídající druhé závorce (subpatternu)
    • flags
      • mění výsledek
      • může mít kombinace (když se dá | mezi hodnoty)
      • PREG_OFFSET_CAPTURE = matches bude obsahovat i offset v bajtech
      • PREG_UNMATCHED_AS_NULL = když např matches[1] by byl “” (nic nenalezeno), tak ho změní na NULL
    • offset
      • od jakého bajtu začne fungovat preg_match

Ukázka matches v preg_match

<?php
preg_match('/(foo)(bar)(baz)/', 'foobarbaz', $matches, PREG_OFFSET_CAPTURE);
print_r($matches);
/* Array
(
    [0] => Array
        (
            [0] => foobarbaz
            [1] => 0
        )

    [1] => Array
        (
            [0] => foo
            [1] => 0
        )

    [2] => Array
        (
            [0] => bar
            [1] => 3
        )

    [3] => Array
        (
            [0] => baz
            [1] => 6
        )

) */

?>

preg_match_all(string $patt, string $subj, array &$matches, int $flags, int offset)

  • vrátí počet shod (včetně 0) či false při chybě
  • hlavička
    • pattern (regex)
      • povinný
    • subject (text)
      • povinný
    • matches
      • uloží shody, struktura záleží na flags
    • flags
      • PREG_PATTERN_ORDER = seskupení podle subpatternů
        • výchozí
        • seskupení dle sloupců
      • PREG_SET_ORDER = seskupení dle řádků
      • PREG_OFFSET_CAPUTRE = matches bude obsahovat i offset v bajtech
      • PREG_UNMATCHED_AS_NULL = když např matches[1] by byl “” (nic nenalezeno), tak ho změní na NULL
    • offset
      • od jakého bajtu začne fungovat preg_match_all

Ukázka matches a flags v preg_match_all

$text = "Jana 25, Petr 30";
$regex = '/([a-zA-Z]+) (\d+)/'; // 1. závorka = Jméno, 2. závorka = Věk

// Seskupí všechna Jména k sobě a všechny Věky k sobě
preg_match_all($regex, $text, $matchesPattern, PREG_PATTERN_ORDER);
print_r($matchesPattern);
/* Výsledek:
Array (
    [0] => Array ( [0] => "Jana 25", [1] => "Petr 30" ), // Všechny celé shody
    [1] => Array ( [0] => "Jana",    [1] => "Petr"    ), // Všechna jména
    [2] => Array ( [0] => "25",      [1] => "30"      )  // Všechny věky
)
*/

// Seskupí data po jednotlivých "osobách"
preg_match_all($regex, $text, $matchesSet, PREG_SET_ORDER);
print_r($matchesSet);
/* Výsledek:
Array (
    [0] => Array (               // 1. osoba (Jana)
        [0] => "Jana 25",
        [1] => "Jana",
        [2] => "25"
    ),
    [1] => Array (               // 2. osoba (Petr)
        [0] => "Petr 30",
        [1] => "Petr",
        [2] => "30"
    )
)
*/

->format('d-m-Y') - naformátuje datum

->modify('+1 day') - úprava času

Řetězové funkce

trim() - smaže bílé znaky ze začátku a konce

toLowerCase() - převede vše na malé písmena

toUpperCase() - převede vše na velké písmena

implode($lepidlo, $pole) - spojí pole do řetězce ((” - ” [a, b]) = “a - b”

explode($sekacek, $string) - převede řetězec na pole (”,”, ”a,b”) = [a,b]
