10. JS - dynamické formuláře, kontrola dat, regulární výrazy

Základní formulářové prvky

<input> - pole vyplňované uživatelem, více forem

<select> a <option>- 1 input více odpovědí

<textarea> - pole pro větší text

<button> - tlačítko

Získání formulářových prvků

.forms['mujFormular'] - podle atributu name formuláře

.getElementById('jmeno') - podle ID

.querySelector('input[name="email"]')- CSS selektor

Získání a změna hodnot

.value - přistupuje k hodnotě, lze přečíst či nahradit

.checked - pro checkboxy a radio, vrací boolean

.disabled - povolí/zakáže prvek

Kontrola dat

  • můžeme zabránit odeslání formůláře díky preventDefaults()
    • nepošle se zbytečný požadavek
    • stránke se nerefreshne
  • stopPropagation() - event neovlivní rodiče

Ukázka

// preventDefaults()
formular.addEventListener("submit", (event) => {
    // 1. Zjistíme chybu
    if (inputJmeno.value === "") {
        // 2. ZASTAVÍME ODESLÁNÍ
        event.preventDefault(); 
        
        // 3. Zobrazíme chybu
        alert("Musíš vyplnit jméno!");
    }
});

divOkno.addEventListener("click", () => {
    zavritOkno();
});



// stopPropagation()
formular.addEventListener("click", (event) => {
    event.stopPropagation(); 
});

Metody

Metody pro RegExp

.test(string) - vrací boolean

.exec(string) - vrací pole s detaily o shodě

Metody pro String

.match(regex) - vrací pole shod

.search(regex) - vrací index první shody

.replace(regex, text2) - nahradí shodu

.split(regex) - rozkouskuje text do pole
