5. Kaskádové styly CSS

Kaskádové styly

Syntax

  • skládá se z
    • hlavička - selector prvku
    • tělo - co to má dělat

Typy implementace CSS

Externí

  • samostatný linknutý soubor .css
  • nejčistší řešení

Interní

  • píše se do hlavičky
  • syntax <style>...</style>
  • proč když to jde oddělit do souboru ¯\_(ツ)_/¯

Inline

  • píše se přímo do tagu
  • syntax <.. style-""> </..>
  • zmrdovské řešení, jen pokud nemáš rád toho, kdo to po tobě má číst

Selector prvku

  • označení tagu
  • možnosti zápisu: (seřazeno sestupně dle váhy, pseudo a tag mají stejnou)
    • id
      • píše se # před název
      • každé id na stránce musí být unikátní
      • př. #hero, #necorandom
    • class
      • píše se . před název
      • znovupoužitelnost
      • př. .navbtn, .card
    • tag
      • označí se pří tag
      • př. h3, nav
    • pseudo
      • n-tý prvky
        • :first-child / :first-of-type
        • :last-child / :last-of-type
        • :nth-child(n) / :nth-of-type(n)
      • stavové
        • :hover - při najetí kurzorem
        • :active - při kliknutí
        • :focus - při zaměření
      • elementy (vytvoří prvek)
        • ::before - vložení před prvek
        • ::after - vložení za prvek
    • kombinace
      • libovolná kombinace předešlých prvků
      • spojovníky
        • A B - potomek B v A
        • A > B - přímý potomek B v A
        • A + B - bezprostřední sourozenec za A
        • A ~ B - sourozenci za A

Vnořené CSS

  • aby se neopakoval kód, můžou se prvky vkládat do sebe
  • některé prohlížeče nepodporují

Ukázka kódů

Stránka je k dispozici zde:

HTML

<!DOCTYPE html>
<html lang="cs">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>CSS Ukázka</title>
    <style>
      body {            
	      margin: 0;            
		    padding: 20px;            
		    background-color: #f0f0f0;        
		   }
    </style>
    <link rel="stylesheet" href="5.css" />
  </head>
  <body>
    <div class="container">
      <div class="card">
        <h3>Karta 1</h3>
        <p>Běžná karta s class selectorem</p>
      </div>
      <div class="card" id="special">
        <h3>Karta 2</h3>
        <p>Druhá karta - nth-child(2n) má gradient</p>
      </div>
      <div style="background-color: antiquewhite;" class="card">
        <h3>Karta 3</h3>
        <p>Třetí karta - běžný styl s menší inline css úpravou</p>
      </div>
    </div>
  </body>
</html>

CSS

/* Class selector */
.container {
  display: flex;
  gap: 20px;
  max-width: 1200px;
  margin: 0 auto;
}
/* Vnořené CSS */
.card {
  width: 300px;
  padding: 20px;
  background-color: white;
  border-radius: 10px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
  position: relative;
  /* Tag selector */
  > h3 {
    margin-top: 0;
  }
}
/* Kombinace - přímý potomek */
.card > p {
  line-height: 1.6;
}
/* Pseudo selector - každá druhá karta */
.card:nth-child(2n) {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
}
/* Pseudo element - ::before */
.card::before {
  content: "★";
  position: absolute;
  top: 10px;
  right: 10px;
  font-size: 24px;
  opacity: 0.3;
}
/* Pseudo state - hover */
.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 8px 16px rgba(0, 0, 0, 0.3);
}
/* ID selector */
#special {
  border: 3px solid #ff6b6b;
}
