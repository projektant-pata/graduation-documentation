= HyperText Markup Language

Struktura souboru

DOCTYPE

  • píše se deklarace HTML dokumentu (čistě HTML, na nic jiného neexistuje)
  • může se specifikovat verze (jinak se vezme nejnovější - HTML5)

HTML

  • kořenový element, obsahuje všechny HTML tagy
  • určuje se jazyk (cs pro češtinu)

HEAD

  • obsahuje info pro prohlížeč

    META - metadata jako kódování stránky (např. UTF-8 pro češtinu)

    TITLE - titulek stránky, zobrazí se v záložce, okně a výsledku hledání

    LINK - připojení externích souborů (např. CSS)

    SCRIPT - připojení JavaScriptu - externě i neexterně (musí se napsat defer v hlavičce, aby se správně spustil)

BODY

  • obsahuje tagy pro komponenty viditelné uživateli

Typy tagů v body

Párové

  • mají konečný tag - <..></..>
  • např. div, nav, header

Nepárové

  • nemají konečný tag - <..>
  • např. img, meta, br

Nepovinně párové

  • konečný tag se může napsat, případně si je prohlížeč automaticky doplní
  • nedobrá praxe - nepřehledný kód a může se to doplnit v špatně (třeba u příkladu)
  • např. p, li

Textové prvky

Nadpisy

  • 6 typů - h1 - h6
  • h1 by se měl vyskytovat jen jednou na stránce
  • důležité pro SEO

Odstavce

  • tag p

sup + sub

  • text, ale s indexováním nahoře (sup) či dole (sub)

span

  • tag span
  • užívá se k obalení části textu

Směrování

Odkazy

  • tag a

Atributy

  • href
    • povinný atribut
    • odkazuje na cílovou destinaci (url, id na stránce..)
  • target
    • nepovinný atribut
    • co nastane po odkazování
    • nejčastěji hodnota _blank (otevře nové okno s cílovou destinací)

Tlačítka (ne na odkazy)

  • tag button

Atributy

  • type
    • jak se chová tlačítko po stisknutí
      • submit - odešle formulář
      • reset - resetuje všechny parametry na výchozí
      • button - nic nedělá, potřebuje JS
  • disabled - znefunkční tlačítko

Seznamy

Seřazený seznam

  • tag ol (ordered list)
  • odrážky budou začínat 1., 2., 3., …

Neseřazený seznam

  • tag ul (unordered list)
  • odrážky začínají stejným symbolem (default je -)

Odrážky

  • tag li
  • musí být v seznamu
