4. HTML formuláře

Formulář

  • posílá data ve formě asociativního pole
  • atributy
    • action
      • = kam se mají data z formu zaslat
      • např. php soubor, metoda mailto:pri@kl.ad
    • method
      • = jak se mají data poslat
      • hodnoty:
        • GET (data jsou v URL)
        • POST (data jsou v těle)
        • dialog

<label>

  • popisek k metadatu (např. inputu)
  • atributy
    • for
      • pro jaké id má odkazovat (ne name!!)
      • po kliknutí se dá focus na prvek s id

<input>

  • atributy
    • type
      • typ vstupu
      • základní hodnoty:
      • povinný atribut

        Datové nevýběrové

        • text
        • password
        • email
        • tel
        • number
        • color
        • date
        • file

        Datové výběrové

        • radio - na výběr z více možností
        • checkbox - true/false
        • range - posuvník

        Ostatní

        • hidden - schová input, ale hodnota se pošle
        • submit - pošle form
        • reset - resetuje všechny inputy na výchozí hodnotu
        • button - nic nedělá, musí se modifikovat přes JS
    • name
      • povinný atribut
      • identifikátor do zaslaného asocativního pole
    • value
      • někdy povinný atribut, např
        • u text nepovinný
        • u radio povinný
      • odeslaná hodnota
    • required
      • nepovinný atribut
      • přidá povinnost vyplnění input
    • placeholder
      • nepovinný atribut
      • výchozí hodnota inputu

<select>

  • = dropdown “input”
  • atributy = input (až na type)
  • obsahuje <option>

<option>

  • jedna z možností <select>

<fieldset> a <legend>

  • užívá se ke seskupení a pojmenování skupin formulářových polí

<fieldset>

  • jedna položka v poli

<legend>

  • název položky

Ukázka

					<fieldset>

	        <br>
	        <br>
	
	        <button id="submit" type="submit">Zaslat</button>
	        <button type="reset">Reset</button>
	    </form>
	</body>
</html>
