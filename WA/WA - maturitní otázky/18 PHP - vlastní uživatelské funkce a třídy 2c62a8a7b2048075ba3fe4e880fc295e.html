18. PHP - vlastní uživatelské funkce a třídy

  • lze deklarovat třídu, metodu i funkci

Funkce

= metoda, ale!

  • neváže se k objektu jako metoda
  • deklarace bez určování modifikátorů dostupnosti

Třída

  • = šablona pro objekt (definuje vlastnosti a metody objektu)
  • objekt je na třídu vázán
  • pokud chceme, aby metoda byla vázána na třídu a ne instanci, dáme jí static
  • Části
    1. vlastnosti
    1. metody
  • funguje OOP

Objekt

= instance třídy

  • definuje data a jak s nimi pracovat
  • deklarace dat
    • lze public private
    • název proměný
    • typ
    • const místo final (jako JS)
//vytvoreni objektu 
$objekt = new Trida()

Metody

// pouziti metody objektu 
$objekt->metoda()

//static u tridy 
Trida::metoda()

Magické metody

  • __construct (konstruktor)
  • __get (pokud vlastnost není public či neexistuje, vrací null)
  • __set (pokud vlastnost není public či neexistuje, vytvoří ji jako public)
  • __toString
    • převede objekt na string
  • __call
    • get, ale pro metody
  • __destruct
    • při ničení objektu

Ukončení metody

  • return
  • dokončeni
  • výjimka

Proměnné

  • píše se před názvem $
  • apresant - přidání reference, ne hodnoty (u typu, co nejsou referenční)
