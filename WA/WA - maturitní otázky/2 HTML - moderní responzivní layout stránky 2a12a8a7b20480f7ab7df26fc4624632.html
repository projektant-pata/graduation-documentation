2. HTML - moderní responzivní layout stránky

Layout stránky

  • dříve přes <table> a float (třeba 20+ let zpátky) !important upravit

Flex

= jednorozměrné pole komponent

  • řeší 1 osu (lze vybrat x nebo y)
  • lehčí na syntax

Responzivní design

  • dělá se přes breakpointy media queries
  • píší se vždy na konec souboru (jinak se přepíšou)
  • styly se aktivují, pokud je podmínka true

Typy podmínek

Dle typu médií

NázevFunkce
all (základní hodnota)vybere všechny typy médií
screenobrazovky (mobily, monitory..)
printpři tisku
speechpři hlasových čtečkách (např. režim pro slepé)

Dle vlastnosti médií

  • jen nejčastější, je jich hafo
NázevFunkce
Rozměry viewportu
width, heighthodnota = ..px
min-width, min-heighthodnota ≥ ..px
max-width, max-heighthodnota ≤ ..px
Ostatní
orientationlandscape x portrait
prefers-color-schemeligth x dark
hoverhover(myš) x none (dotyk)

Možnosti spojování podmínek

  • (min-width: ..px) - jednoduchá (pokud šířka obrazovky překročí ..px, použije se)
  • (min-width: ..px) and (max-width: ..px) - spojení AND (rozmezí mezi šířkami obrazovky)
  • (max-width: ..px), (min-width: ..px) - nebo OR (pokud je obrazovka do ..px nebo od ..px)
  • screen not (hover: none) - bez NOT (obrazovka, pokud není dotyková)

!important mobile first

Ukázka

Stránka je k dispozici zde:

<!DOCTYPE html>
<html lang="cs">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Flex a Grid - Jednoduchá ukázka</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        body {
            font-family: Arial, sans-serif;
            padding: 20px;
        }

        h2 {
            margin: 20px 0 10px;
        }

        /* ===== FLEXBOX ===== */
        .flex-container {
            display: flex;
            gap: 10px;
            background: #1976d2;
            padding: 10px;
        }

        .flex-item {
            background: #fff;
            padding: 20px;
            text-align: center;
            flex: 1;
        }

        /* ===== GRID ===== */
        .grid-container {
            display: grid;
            gap: 10px;
            background: #7b1fa2;
            padding: 10px;
        }

        .grid-item {
            background: #fff;
            padding: 20px;
            text-align: center;
        }

        /* ===== MOBILE FIRST - BASE STYLY ===== */
        /* Mobil - 1 sloupec */
        .flex-container {
            flex-direction: column;
        }

        .grid-container {
            grid-template-columns: 1fr;
        }

        /* ===== BREAKPOINTY ===== */
        /* Tablet (768px+) - 2 sloupce */
        @media (min-width: 768px) {
            .flex-container {
                flex-direction: row;
            }

            .grid-container {
                grid-template-columns: repeat(2, 1fr);
            }
        }

        /* Desktop (1024px+) - 3 sloupce */
        @media (min-width: 1024px) {
            .grid-container {
                grid-template-columns: repeat(3, 1fr);
            }
        }
    </style>
</head>
<body>
    <h2>Flexbox</h2>
    <div class="flex-container">
        <div class="flex-item">Flex 1</div>
        <div class="flex-item">Flex 2</div>
        <div class="flex-item">Flex 3</div>
    </div>

    <h2>Grid</h2>
    <div class="grid-container">
        <div class="grid-item">Grid 1</div>
        <div class="grid-item">Grid 2</div>
        <div class="grid-item">Grid 3</div>
        <div class="grid-item">Grid 4</div>
        <div class="grid-item">Grid 5</div>
        <div class="grid-item">Grid 6</div>
    </div>
</body>
</html>

Bajer hund

Kdy se využije min-width a kdy max-width?

Záleží na typu vývoje, zda je mobile-first a nebo desktop-first

Moderní standart je dělat napřed stránku pro mobil (mobile-first), protože od roku 2016 se většina stránek otevře na mobilním zařízení, ale je možnost i zastaralejšího a častějšího způsobu - napřed dělat stránku pro PC, když jde o desktop focused aplikaci. Jinak lze použít obě najednou či range)
