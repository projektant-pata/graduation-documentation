  • nutnost pořadí + vše lze napsat jen jednou (až na JOIN)

SELECT

  • slouží pro výpis dat
  • nutnost napsat, která data chceme vrátit (můžeme je pojmenovat přes AS, zároveň to ulehčí budoucí práci)
  • nutnost napsat hlavní tabulku (FROM)
  • DISTINCT - odstraňuje duplicity

Příklad syntaxe:

SELECT DISTINCT jmeno, vek AS aktualni_vek
FROM studenti;

JOIN

  • spojí tabulky
  • podmínka ON (něco jako WHERE)

INNER JOIN mesta ON studenti.mesto_ID = mesta.mesto_ID;

WHERE

  • redukuje záznamy na ty, co splňují podmínku

Příklad syntaxe s WHERE

SELECT DISTINCT studenti.jmeno AS jmeno, studenti.vek AS vek, mesta.nazev AS trvale_bydliste
FROM studenti 
NATURAL INNER JOIN mesta
WHERE mesta.nazev != 'Veska';

GROUP BY a HAVING

  • seskupí záznamy a vyfiltruje

Příklad syntaxe s GROUP BY s výběrem 

SELECT studenti.vek AS vek, COUNT(*) AS pocet_studentu
FROM studenti
INNER JOIN mesta ON studenti.mesto_ID = mesta.mesto_ID
WHERE mesta.nazev != 'Veska'
GROUP BY studenti.vek
HAVING COUNT(*) >= 3
ORDER BY studenti.vek;

Výsledek

vekpocet_studentu
183

ORDER BY

  • seřadí záznamy

Příklad syntaxe s GROUP BY s výběrem 

SELECT studenti.vek AS vek, COUNT(*) AS pocet_studentu
FROM studenti
INNER JOIN mesta ON studenti.mesto_ID = mesta.mesto_ID
WHERE mesta.nazev != 'Veska'
GROUP BY studenti.vek
HAVING COUNT(*) >= 1
ORDER BY COUNT(*) ASC, studenti.vek DESC;

Výsledek

191
141

LIMIT

  • vrátí určitý počet

Příklad syntaxe s GROUP BY s výběrem 

SELECT studenti.vek AS vek, COUNT(*) AS pocet_studentu
FROM studenti
INNER JOIN mesta ON studenti.mesto_ID = mesta.mesto_ID
WHERE mesta.nazev != 'Veska'
GROUP BY studenti.vek
HAVING COUNT(*) >= 1
ORDER BY COUNT(*) ASC, studenti.vek DESC
LIMIT 2;

Matematické

  • ROUND(cislo) - zaokrouhlí
  • FLOOR(cislo) - zaokrouhlí dolů
  • CEILING(cislo) - zaokrouhlí nahoru
  • TRUNCATE(cislo) - smaže desetinné místo (FLOOR()u kladných čísel, CEILING() u negativních čísel)
  • ABS(cislo) - absolutní hodnota
  • POWER(cislo) - pro mocninu
  • SQRT(cislo) - pro odmocninu

Stringové

  • CONCAT(str1, ‘ - ’, str2) - spojuje řetězce
  • LENGTH(str1)
  • UPPER() - VŠECHNO VELKÝM
  • LOWER() - všechno malým
  • TRIM(), LTRIM(), RTRIM()

Datumové

  • DATE_FORMAT(date, regex) - zformátuje datum
    • příklad regexu ‘%d. %m. %Y’ - !!jenom 1 písmenko, zbytek se tam proste napíše
  • DAY()
  • MONTH()
  • YEAR()
