20. PHP - soubory

Nahrávání souboru

Co je potřeba?

  • metoda musí být POST
  • formulář musí obsahovat enctype="multipart/form-data"
    • bez toho atributu se pošle jen název souboru
  • input typu file

Přístup k souborům

  • v 2-rozměrném asociativním globálním poli $_FILES

Obsah nahraného souboru

  • [‘name’]
    • původní název včetně přípony
    • např. fotkatvojimamky.jpg
    • ! může obsahovat nebezpečné znaky (např. 卍)
  • [’tmp_name’]
    • cesta k dočasnému soboru uloženému na serveru
    • např. C:\WINDOWS\TEMP\fotkatvojimamky.jpg
    • pro trvalé uložení souboru je nutné ho přesunout z dočasné cesty, po dokončení skriptu se smaže
  • [‘size’]
    • velikost nahraného souboru v bajtech
    • např. 10240 (10kB)
  • [’type’]
    • MIME typ souboru
    • např. image/png, application/pdf
    • ! informace není ověřená serverem, útočník může skrýt skript za image/jpeg
  • [’error’]
    • číselný kód informující o stavu nahrání
    • hodnoty (0-8)
      • 0 (UPLOAD_ERR_OK) - vše v pořádku, soubor nahrán
      • 1 (UPLOAD_ERR_INI_SIZE) - soubor je větší než definovaný limit v php.ini
      • 2 (UPLOAD_ERR_FORM_SIZE) - soubor je větší než definovaný limit v HTML formuláři
      • 4 (UPLOAD_ERR_NO_FILE) - žádný soubor nebyl nahrán

Metody pro práci se soubory

Kontrolní metody

mime_type_content(): string - ověří typ souboru na straně serveru, takže nejde podvrhnout

file_exist($cesta): bool - ověří, zda soubor existuje

is_dir($cesta): bool - ověří, zda existuje adresář

Práce se soubory

move_uploaded_file($odkud, $cum): bool - přesune soubor (bezpečnější jak rename)

rename($staraCesta, $novaCesta): bool - přejmenuje soubor (pokud se liší cesta, tak ho i přesune)

copy($source, $dest): bool - zkopíruje soubor

unlink($cesta): bool - smaže soubor

Otevírání a čtení souborů

fopen($cesta, $rezim): resource - vytvoří handler pro práci se souborem (režimy ‘r’ - čtení, 'w' - zápis/přepis, 'a' - přepis na konec (append)

fclose($resource): bool - zavře handler

Práce s adresáři

  • mkdir($cesta): bool - vytvoří adresář
  • rmdir($cesta): bool - odstraní adresář
  • opendir($cesta): resource - vytvoří handler pro práci s adresářem
  • readdir($handle): string - vrátí název souboru/složky
  • closedir($handle): void - uzavře handler
  • scandir($cesta): array - vrací pole souborů a složek

Ukázka

<form action="upload.php" method="POST" enctype="multipart/form-data">
    Vyberte soubor: <input type="file" name="soubor">
    <input type="submit" value="Nahrát">
</form>
<?php
$slozka = "uploads/";

if (is_dir($slozka)) {
    $soubory = scandir($slozka);

    echo "<div class='galerie'>";

    foreach ($soubory as $soubor) {
        if ($soubor === '.' || $soubor === '..') {
            continue;
        }

        $cesta = $slozka . $soubor;
        $pripona = strtolower(pathinfo($cesta, PATHINFO_EXTENSION));

        echo "<div class='polozka' style='margin: 10px; border: 1px solid #ccc; padding: 10px;'>";
        
        if (in_array($pripona, ['jpg', 'jpeg', 'png', 'gif'])) {
            echo "<img src='$cesta' alt='$soubor' style='max-width: 150px; display: block;'>";
        } 

        echo "</div>";
    }

    echo "</div>";
} else {
    echo "Složka neexistuje.";
}
?>
